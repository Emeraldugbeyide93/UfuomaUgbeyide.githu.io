SENTIMENT ANALYSIS IN PYTHON

In this project, a sentiment analysis was performed using Bank of Scotland's mobile applications customer reviews which were obtained from google play store. The key libraries used in this project included Pandas, Numpy, Seaborn, Matplotlib, WordCloud and Sklearn amongst others which were also used to carry out some data cleaning, lemmatization, stemming and exploration tasks. Gradient Boosting, Support Vector Machine and XGBoost. A 10-Fold cross validation was also carried out on results and a model performance comparison was carried out at the end using the AUC-ROC score.

SENTIMENT CLASSIFICATION USING VADER

In this project, customer reviews which were obtained from Bank of Scotland were classified into the different sentiment classes (Positive, Negative and Neutral) based on their polarity scores using VADER (Valence Aware Dictionary and Sentiment Reasoner) . The libraries used in this project were Pandas, Numpy, Matplotlib, NLTK.
